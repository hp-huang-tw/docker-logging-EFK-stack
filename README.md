# docker-logging-EFK-stack
docker-compose starter for fluentd with EFK Stack for Docker logging

## Environment Setting
- Elasticsearch version: 6.2.4
- Fluentd version: 0.12
- Kibana version: 6.2.4

## Get Started
```
docker-compose up
```
